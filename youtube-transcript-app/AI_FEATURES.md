## üìã How to Use

### Step 1: Unlock AI Features
1. Go to **Settings**
2. Scroll to **AI Features Configuration**
3. Choose your provider (Groq or Gemini)
4. Enter access codes OR your custom API key
5. Click "Unlock"

### Step 2: Fetch a Transcript
1. Go to **Fetch Transcript**
2. Paste a YouTube URL
3. Click "Fetch"
4. Wait for transcript to load

### Step 3: Use AI Tools
1. Navigate to **AI Tools** tab
2. Select a tool from the grid
3. Configure options (if any)
4. Click the action button
5. View results
6. Copy to clipboard if needed
